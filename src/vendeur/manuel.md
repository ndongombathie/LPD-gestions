1. Connexion à l'application
Avant d'accéder à l'interface de vente, vous devez vous authentifier.

Page de connexion
Image 1 : (Insérez ici une capture d'écran de la page de connexion)

Description de l'image : La page affiche un fond coloré (dégradé violet/orange). Au centre, un rectangle blanc translucide contient le logo "LPD" en grand, suivi du titre "Connexion au tableau de bord". En dessous, vous voyez deux champs de saisie :

Un champ "Adresse e-mail"

Un champ "Mot de passe" avec une icône en forme d'œil pour afficher/masquer le texte.

Un bouton "Se connecter" coloré.

En bas, le copyright "© 2025 LPD Entreprise".

Procédure :

Saisissez votre adresse e-mail dans le premier champ.
Saisissez votre mot de passe dans le second champ. Vous pouvez cliquer sur l'icône en forme d'œil (👁️) pour vérifier ce que vous tapez.
Cliquez sur le bouton "Se connecter".
Gestion des erreurs
Image 2 (optionnelle) : (Insérez ici une capture d'écran avec un message d'erreur)

Description de l'image : La même page de connexion, mais avec un petit message en rouge (ex: "❌ Identifiants incorrects") apparaissant sous le bouton. Une notification "toast" (un petit rectangle qui apparaît en haut à droite) est également visible en rouge avec le message "Échec de connexion".

Explication : Si vous oubliez un champ ou que vos identifiants sont incorrects, un message d'erreur s'affichera pour vous guider.

2. Vue d'ensemble de l'interface
Après une connexion réussie, vous arriverez sur l'interface principale.

Image 3 : (Insérez ici une capture d'écran de l'interface principale après connexion)

Description de l'image : L'écran est divisé en trois zones :

À gauche, une barre latérale violette avec le logo "LPD" et trois liens : "Tableau de bord", "Nouvelle commande", "Historique". L'option "Tableau de bord" est surlignée.

En haut, une bande de couleur (dégradé violet/orange) suivie d'une barre d'en-tête blanche. À droite de cette barre, on voit les initiales de l'utilisateur (ex: "JD") et son nom (ex: "Jean Dupont").

Au centre, le contenu principal : le tableau de bord avec des cartes de statistiques.

La Barre Latérale (Menu)
Fonction : Elle vous permet de naviguer entre les différentes sections de l'application.

Éléments :

Tableau de bord : La page d'accueil avec un résumé de votre activité.

Nouvelle commande : L'endroit pour créer une nouvelle vente.

Historique : La liste de toutes les commandes passées.

Sur mobile : Le menu est masqué par défaut. Vous pouvez l'ouvrir en cliquant sur l'icône "burger" (☰) dans l'en-tête.

L'En-tête (Header)
Image 4 : (Insérez ici un zoom sur la partie droite de l'en-tête, avec le menu utilisateur ouvert)

Description de l'image : On voit le nom de l'utilisateur, ses initiales dans un cercle, et une petite flèche vers le bas. En cliquant dessus, un menu déroulant blanc apparaît. Ce menu contient l'email de l'utilisateur, le nom de la boutique, une option "Changer mot de passe" et une option "Déconnexion" en orange.

Informations :

À gauche, le nom de l'application ("LPD Manager | Interface Vendeur").

Au centre, le montant des "Ventes du jour" s'affiche et se met à jour automatiquement.

Menu utilisateur :

Cliquez sur votre nom et vos initiales (en haut à droite) pour ouvrir un menu déroulant.

Ce menu affiche votre nom, votre email et le nom de votre boutique.

Vous y trouverez les options pour "Changer mot de passe" et "Déconnexion".

3. Tableau de bord
Le tableau de bord vous donne un aperçu instantané de votre activité.

Image 5 : (Insérez ici une capture d'écran de la page "Tableau de bord" en entier)

Description de l'image : Sous l'en-tête, on voit un titre "Tableau de bord vendeur". À côté, la date du jour et un bouton "Rafraîchir". Ensuite, deux grandes cartes : "Ventes du jour" avec un montant en grosses lettres, et "Commandes du jour" avec un nombre. En dessous, une section "Dernières commandes" liste 3 ou 4 commandes avec le nom du client, le statut (pastille verte, jaune ou rouge) et le montant.

Ventes et Commandes du Jour
Deux cartes principales résument la journée en cours :

Ventes du jour : Affiche le montant total des ventes (commandes complétées) d'aujourd'hui.

Commandes du jour : Affiche le nombre total de commandes passées aujourd'hui, avec un détail visuel des statuts (complétées en vert, en attente en jaune, annulées en rouge).

Dernières Commandes
Cette section liste les 4 dernières commandes enregistrées.

Pour chaque commande, vous voyez le client, le statut (Complétée, En attente, etc.), le montant total et l'heure.

Cela vous permet de vérifier rapidement les transactions les plus récentes.

4. Nouvelle Commande
C'est le cœur de votre activité. Voici comment créer une commande étape par étape.

Image 6 : (Insérez ici une capture d'écran de la page "Nouvelle commande" vierge, avec la zone de gauche pour les produits et la zone de droite vide pour le panier)

Description de l'image : La page est divisée en deux grandes parties. À gauche, le titre "Nouvelle Commande" et plusieurs sections : "Type de Vente Global" avec deux boutons (Détail/Gros), "Scanner Code Barre" avec un champ de saisie, "Recherche Produits" avec un champ de saisie, et enfin une grille de produits disponibles. À droite, une zone intitulée "Panier" est pour l'instant vide, avec l'icône d'un panier et le texte "Panier vide".

1. Choisir le type de vente global
Image 7 (zoom) : (Capture d'écran de la section "Type de Vente Global", avec le bouton "Détail" surligné en violet et le bouton "Gros" en orange)

Explication : En haut de la section de gauche, choisissez entre "Détail" (vente à l'unité, prix en violet) et "Gros" (vente par carton, prix en orange). Le type choisi déterminera le prix appliqué par défaut quand vous ajouterez un produit.

2. Scanner un code-barres
Image 8 (zoom) : (Capture d'écran de la section "Scanner Code Barre", avec le champ de saisie vide et le bouton "Scanner" à côté)

Procédure :

Placez le curseur dans le champ "Scannez ou entrez le code barre...".
Utilisez votre douchette pour scanner le code-barres d'un produit, ou saisissez-le manuellement.
Appuyez sur la touche "Entrée" de votre clavier ou cliquez sur le bouton "Scanner".
Le produit sera automatiquement ajouté à votre panier.
3. Rechercher un produit
Image 9 (zoom) : (Capture d'écran de la section "Recherche Produits", avec le mot "cahier" tapé dans le champ et les résultats filtrés en dessous)

Procédure :

Dans le champ "Nom, code barre ou catégorie...", tapez le nom du produit, une partie de son nom, ou sa catégorie (ex: "cahier", "stylo", "ramette").
La liste des produits en dessous se filtrera automatiquement au fur et à mesure que vous tapez.
4. Sélectionner un produit
Image 10 (zoom) : (Capture d'écran de la grille de produits, avec un produit spécifique mis en avant. On voit son nom, son code-barres, sa catégorie, son prix de détail, son prix de gros, et deux boutons "Détail" et "Gros")

Description : Chaque produit dans la liste s'affiche sous forme de petite carte. Vous voyez :

Le nom du produit.

Son code-barres (s'il existe).

Sa catégorie.

Son prix de détail en vert, avec le prix seuil en dessous (prix minimum autorisé).

Son prix de gros en vert, avec le prix seuil en dessous.

Procédure :

Pour ajouter le produit au panier en tant que vente à l'unité, cliquez sur le bouton "Détail" (violet).

Pour ajouter le produit au panier en tant que vente en gros, cliquez sur le bouton "Gros" (orange).

5. Renseigner les informations client
Image 11 (zoom) : (Capture d'écran de la zone "Informations Client" à droite, avec les champs "Nom", "Prénom", "Téléphone", "Adresse")

Procédure :

Nom * (obligatoire) : Saisissez le nom de famille du client.
Prénom * (obligatoire) : Saisissez le prénom du client.
Téléphone (optionnel) : Saisissez le numéro de téléphone.
Adresse (optionnel) : Saisissez l'adresse complète du client.
Une fois le nom et le prénom saisis, un encart vert "Client identifié" apparaît pour confirmation.

6. Gérer le panier
Image 12 : (Insérez ici une capture d'écran de la zone de droite "Panier" avec plusieurs produits ajoutés, afin de montrer toutes les interactions possibles)

Description : Le panier est maintenant rempli. On voit les produits regroupés par type (Détail / Gros). Pour chaque produit, on peut voir son nom, son code-barres, sa catégorie, le prix unitaire et la quantité, ainsi que le total pour cette ligne.

Modifier la quantité
Image 13 (zoom) : (Focus sur les boutons "+" et "-" et le chiffre de la quantité d'un produit dans le panier)

Utilisez les boutons "+" et "-" pour augmenter ou diminuer la quantité.

Vous pouvez aussi cliquer directement sur le chiffre de la quantité. Une petite fenêtre s'ouvrira pour vous permettre de taper la quantité exacte.

Modifier le prix unitaire
Image 14 (zoom) : (Focus sur l'icône en forme de crayon (modifier) à côté d'un produit dans le panier, et sur la petite fenêtre qui s'ouvre pour changer le prix)

Cliquez sur l'icône "crayon" (✏️) à côté du prix.

Une petite fenêtre jaune apparaît. Saisissez le nouveau prix (il ne doit pas être inférieur au "prix seuil" indiqué).

Validez avec le bouton vert (✓) ou annulez avec la croix rouge (✗).

Retirer un produit
Cliquez sur l'icône "poubelle" (🗑️) à côté du produit que vous souhaitez retirer.

7. Valider la commande et imprimer le ticket
Image 15 (zoom) : (Capture d'écran du bas de la zone de droite, montrant le "Résumé de la commande" avec les totaux, et les deux boutons "Envoyer la Commande" (vert) et "Annuler Commande" (rouge))

Résumé : En bas du panier, un récapitulatif affiche le Total HT, la TVA (si activée via la case à cocher), et le Total TTC en gros et en vert.

Procédure :

Vérifiez que le panier et les informations client sont corrects.
Cliquez sur le bouton vert "Envoyer la Commande".
La commande est enregistrée.
Image 16 : (Insérez ici une capture d'écran du ticket de caisse qui s'affiche à l'écran après la validation)

Description : Une fenêtre apparaît avec un aperçu du ticket de caisse. On y voit le logo LPD, la date, le nom du vendeur, le nom du client, un QR code, et le numéro de commande.

Procédure :

Cliquez sur le bouton "Imprimer" dans cette fenêtre pour imprimer le ticket sur l'imprimante thermique.
Cliquez sur "Fermer" pour revenir à l'écran de nouvelle commande (le panier sera vidé, prêt pour une nouvelle vente).
5. Historique des commandes
Cette page vous permet de consulter toutes les commandes passées.

Image 17 : (Insérez ici une capture d'écran de la page "Historique des commandes")

Description de l'image : La page a un titre "Historique des Commandes". En haut, on voit des petites cartes de statistiques (Aujourd'hui, En attente, Validées). Ensuite, il y a une barre avec l'icône "Filtres". En dessous, une longue liste de commandes s'affiche, chacune avec son numéro, son statut, le nom du client, la date et le montant.

Filtres et recherche
Image 18 (zoom) : (Capture d'écran de la section des filtres déroulée, montrant les options pour le Statut, le Type, la Période et la Recherche)

Cliquez sur la barre "Filtres" pour dérouler les options de recherche avancée.

Vous pouvez filtrer les commandes par statut (tous, complétées, en attente, annulées), par type de vente (détail, gros, mixte), par période (aujourd'hui, hier, 7 jours, etc.), ou faire une recherche textuelle par numéro de commande ou nom de client.

Liste des commandes
Chaque ligne de la liste donne un aperçu : numéro de commande, statut (avec une pastille de couleur), type de vente, nom du client, téléphone, date, et montant total.

Cliquez sur le bouton "Détails" (bleu) pour voir les informations complètes d'une commande.

Détails d'une commande
Image 19 : (Insérez ici une capture d'écran de la fenêtre modale "Détails de la commande" qui s'ouvre après avoir cliqué sur le bouton "Détails")

Description de l'image : Une fenêtre blanche s'ouvre par-dessus la liste. Elle contient trois sections :

Informations (statut, type, date, vendeur).

Client (nom, téléphone, adresse).

Produits (un tableau listant les produits achetés, avec leur type, quantité, prix unitaire et sous-total).

En bas, le récapitulatif financier (Total HT, TVA, Total TTC).

6. Gestion du compte utilisateur
Ces actions se passent dans le menu en haut à droite.

Changer son mot de passe
Image 20 : (Insérez ici une capture d'écran de la fenêtre modale "Changer le mot de passe")

Description de l'image : Une petite fenêtre blanche avec le titre "Changer le mot de passe". Elle contient trois champs : "Ancien mot de passe", "Nouveau mot de passe", "Confirmer le mot de passe". En bas, deux boutons "Annuler" et "Confirmer".

Procédure :

Dans le menu utilisateur, cliquez sur "Changer mot de passe".
Dans la fenêtre qui s'ouvre, saisissez votre ancien mot de passe.
Saisissez votre nouveau mot de passe (minimum 6 caractères).
Confirmez le nouveau mot de passe.
Cliquez sur "Confirmer". Un message de succès vert apparaîtra si tout est correct.
Se déconnecter
Dans le menu utilisateur, cliquez sur "Déconnexion".

Confirmez votre choix dans la boîte de dialogue qui apparaît. Vous serez redirigé vers la page de connexion.
